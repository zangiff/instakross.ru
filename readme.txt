Лендинг, полный нестандартных задач по верстке, связанных с особенностями дизайна.
Кроссбраузерная верстка с адаптивом, с поддержкой Сафари и Инстаграм браузера.
Из особенностей - видео в слайдере с кастомной стилизацией и подключением через тег video.
Без Bootstrap и каких-либо еще фреймворков.
Только исходники, без сборки и видеофайлов в папке ./app/video
